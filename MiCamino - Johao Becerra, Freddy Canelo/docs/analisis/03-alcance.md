# Alcance del Proyecto

El presente proyecto tiene como objetivo desarrollar una plataforma web sencilla, práctica y accesible que oriente a estudiantes de grados décimo y once de instituciones educativas públicas de Medellín en su proceso de planificación profesional. La solución se implementará utilizando tecnologías básicas y de libre acceso como HTML, CSS y JavaScript, garantizando facilidad de uso y compatibilidad con dispositivos de baja gama.

**La plataforma constará de dos secciones principales:**

**1. Página de registro y cuestionario inicial:**
Un espacio donde los estudiantes podrán crear un perfil personal. Se incluirán preguntas orientadas a explorar sus intereses académicos, habilidades, conocimientos adquiridos y algunos datos sociodemográficos básicos. Esto permitirá generar un perfil individualizado que sirva como punto de partida para la orientación profesional.

**2. Página de resultados y orientación:** 
Basada en las respuestas y el análisis interno, el sistema mostrará porcentajes de afinidad con diferentes áreas académicas, sugerencias de carreras técnicas, tecnológicas o universitarias y ejemplos de ofertas laborales vigentes asociadas a cada campo. Además, brindará información sobre instituciones locales donde se pueden cursar dichas carreras.

Este proyecto no incluye el desarrollo de aplicaciones móviles nativas ni la integración con bases de datos complejas en la primera fase. Tampoco abordará la generación de rutas de aprendizaje personalizadas más allá de la orientación inicial, ni la conexión directa con empresas reclutadoras. Su alcance se limita a una plataforma web informativa e interactiva de bajo costo de implementación, diseñada para apoyar procesos de orientación vocacional en colegios públicos.

**Las restricciones incluyen:**

- Dependencia de datos públicos y disponibles en línea, como observatorios laborales, plataformas del Ministerio de Educación y reportes del DANE.
- Recursos técnicos limitados a lenguajes front-end (HTML, CSS, JavaScript), lo que implica que la plataforma será funcional pero sin sistemas avanzados de inteligencia artificial en la primera fase.
- Acceso condicionado a la conectividad de los estudiantes, dado que el proyecto estará disponible únicamente vía web.
