# OBJETIVOS ESPECÍFICOS

- Diagnosticar las necesidades de orientación vocacional de los estudiantes de grados décimo y once en colegios públicos de Medellín, mediante encuestas y entrevistas a estudiantes, docentes y orientadores.

- Analizar datos secundarios del mercado laboral local — provenientes del **OLE**, el **DANE**, **LinkedIn** y la **UNESCO** — para identificar los sectores emergentes y las competencias más demandadas en Medellín.

- Diseñar la arquitectura y los componentes del prototipo web, incluyendo el módulo de registro de usuario, el cuestionario de habilidades e intereses, y la página de resultados con perfil vocacional sugerido.

- Desarrollar el prototipo funcional utilizando tecnologías de libre acceso **(HTML, CSS y JavaScript)**, garantizando compatibilidad con dispositivos de baja gama y accesibilidad para estudiantes con conectividad limitada.

- Validar el prototipo con usuarios reales del contexto educativo público de Medellín, evaluando su usabilidad, pertinencia de los resultados y utilidad percibida para la toma de decisiones vocacionales.
